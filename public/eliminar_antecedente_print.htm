Eliminar antecedente

En este capítulo se explica cómo eliminar un antecedente:
 

Nombre del factor a eliminar

Se muestra un campo de texto con el nombre del factor a eliminar, cabe resaltar que la eliminación es de manera lógica, es decir que no se elimina de la base de datos, sino solo se oculta a la vista del usuario. Ademas, si este factor ha sido usado para la creaacion de una regla no puede ser eliminado. Arriba

Cancelar eliminación del factor

Seleccionar este botón para cancelar la eliminación del factor. Arriba

Confirmar eliminación del factor

Seleccionar este boton para confirmar la eliminación del factor. Arriba
