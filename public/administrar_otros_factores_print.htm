Administrar otros factores

En este capítulo se explica cómo administrar otros factores:
 
  • Nuevo factor
  • Editar factor
  • Eliminar factor
 
 

Seleccionar pestaña de otros factores

Seleccionar esta pestaña para acceder a la lista de otros factores, donde se puede editar y eliminar cada uno de los otros factores. Arriba

Editar un factor

Seleccionar este botón para editar un factor registrado, los campos que se podrán editar son los siguientes, Nombre, Descripción e Imagen. Arriba

Eliminar un factor

Seleccionar este botón un factor registrado, aparecera una ventana emergente para confirmar la eliminación. Arriba

Registrar un nuevo factor

Seleccionar este boton para registrar un nuevo factor, aparece una ventana emergente con varios campos para registrar un nuevo factor. Arriba
