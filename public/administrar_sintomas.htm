Administrar sintomas

En este capítulo se explica cómo administrar los síntomas:
 
  • Nuevo síntoma
  • Editar síntoma
  • Eliminar síntoma
 
 

Seleccionar pestaña de síntomas

Seleccionar esta pestaña para acceder a la lista de antecedentes, donde se puede editar y eliminar cada una de los antecedentes. Arriba
Arriba

Editar un síntoma

Seleccionar este botón para editar un síntoma registrado, los campos que se podrán editar son los siguientes: Nombre, Descripción e Imagen. Arriba
Arriba

Eliminar un síntoma

Seleccionar este botón para eliminar un síntoma registrado, aparecerá una ventana emergente para confirmar la eliminación. Arriba
Arriba

Registrar un nuevo síntoma

Seleccionar este botón para registrar un nuevo síntoma, aparecerá una ventana emergente con varios campos para registrar un nuevo síntoma. Arriba
