2. Iniciar sesión

Es la primera pantalla que se observa del software, esta pantalla permite iniciar sesión para acceder al sistema experto, en este caso el sistema internamente habilitara los permisos de administrador, empleado y visitante, estos roles serán explicados en este apartado 2.1. Roles y permisos.
 
 

Ingresar Email

Seleccionar este campo para ingresar el email del trabajador que en este sistema actua como su usuario para acceder al sistema. Arriba

Ingresar contraseña

Seleccionar este campo para ingresar la contraseña del trabajador, cabe resaltar que la contraseña es creada por el administrador, ya que éste es el que administra todos los usuarios. Arriba

Ingresar al sistema

Seleccionar este boton para acceder al sistema, internamente el sistema evaluará a que tipo de usuario pertenece y otorgará permisos. Arriba
