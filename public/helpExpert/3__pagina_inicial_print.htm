3. Página inicial

Esta pagina es la primera pantalla que se observa del sistema experto web.
 
 

Módulo de pacientes

En este módulo se tendrá acceso registrar los pacientes, asi como a administrarlos. Arriba

Módulo de factores

En este módulo se administran los tipos de factores que darían pistas que una persona tiene una determinada enfermedad, en este apartado no solo se verán los sintomas, sino también los antecedenetes, asi como los diferentes factores que podrian desencadenar una enfermedad. Arriba

Módulo de enfermedades

Este módulo contiene la administración de enfermedades, por ahora solo son tres: Dengue y su variante Dengue Hemorrágico, Chikunguya y Zika. Arriba 

Módulo de recomendaciones

Este módulo se encargara de a administración de las recomendaciones, dentro de estas recomendaciones están los examenes posteriores a la evaluación que se deben hacer al paciente, asi como tambien las buenas prácticas que debe realizar el paciente para mejorar su condición. Arriba

Módulo de reglas de conocimiento

Este módulo es uno de las importantes, porque permite ingresar las nuevas reglas de conocimiento que se crean necesarias, se toman en cuenta varios factores para decidir la creación o eliminacon de una regla de conocimiento, ya que esto es la materia prima del sistema experto, si no esta creado correctamente el sistema no arrojará los resultados esperados. Arriba

Módulo de diagnóstico

Este módulo es el mas importante, ya que permite el diagnóstico de las enfermedades usando las reglas de conocimiento anteriormente registradas, asi tambien se visualiza la confirmación del resultado al no estar 100%, el sistema empecera a repreguntar por sintomas que aun falten ingresar. Arriba

Módulo de Usuarios

Este módulo permite la administración de los usuarios, por parte de los usuarios tipo administrador. Arriba
