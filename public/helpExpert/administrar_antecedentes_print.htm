Administrar antecedentes

En este capítulo se explica cómo administrar los antecedentes:
 
  • Nuevo antecedente
  • Editar antecedente
  • Eliminar antecedente
 

Seleccionar pestaña de antecedentes

Seleccionar esta pestaña para acceder a la lista de antecedentes, donde se puede editar y eliminar cada una de los antecedentes.

Editar un antecedente

Seleccionar este botón para editar un antecedente registrado, los campos que se podrán editar son los siguientes: Nombre, Descripción e Imagen. Arriba

Eliminar un antecedente

Seleccionar este botón para eliminar el antecedente registrado, aparecerá una ventana emergente para confirmar la eliminación. Arriba

Registrar un nuevo antecedente

Seleccionar este botón para registrar un nuevo antecedente, aparece una ventana emergente con varios campos para registrar un nuevo antecedente. Arriba
