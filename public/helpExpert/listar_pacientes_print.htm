Listar Pacientes

En este capítulo se explica cómo listar los pacientes:
 

Lista de los pacientes registrados por el sistema

Esta tabla muestra los pacientes registrados en el sistema, cada uno tiene una opción de editar o eliminar. Arriba

Registrar nuevo paciente

Seleccionar este botón si se desea registrar un nuevo paciente, al momento de iniciar la consulta. Arriba

Editar el registro de un paciente

Seleccionar este botón si se desea editar los datos, se muestra una ventana emergente para editar los datos del paciente

Eliminar el registro de un paciente

Seleccionar este botón si se desea eliminar los datos de un paciente, se muestra una ventana emergente para confirmar la eliminación. Arriba
