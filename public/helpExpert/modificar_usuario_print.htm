Modificar usuario

En esta sección se puede modificar un usuario antes registrado.
 

Email de usuario

En este campo se ingresa el nuevo email de usuario o en su defecto se deja como está, se debe tener en cuenta que no se pueden registrar dos usuarios con el mismo email. Arriba

Contraseña de usuario

En este campo se debe ingresar la nueva contraseña del usuario. Arriba

Tipo de usuario

En este combo se debe seleccionar el tipo de usuario. Arriba

Cancelar modificación de usuario

Seleccione este botón si desea cancelar la modificación del usuario. Arriba

Guardar usuario

Seleccione esta opción si desea guardar los datos modificados del usuario. Arriba
