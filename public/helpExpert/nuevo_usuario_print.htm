Nuevo usuario

En esta sección se puede agregar nuevos usuarios al sistema experto.

Email de usuario

En este campo se debe ingresar el correo electrónico del usuario, se debe tener en cuenta que no se puede registrar 2 usuarios con la misma cuenta de correo electrónico. Arriba

Contraseña de usuario

En este campo se debe ingresar la contraseña del usuario. Arriba

Tipo de usuario

En este combo se debe seleccionar el tipo de usuario, para que según eso tenga los permisos, de eliminar, crear o visualizar únicamente. Arriba

Cancelar creación de usuario

Seleccionar este boton si desea cancelar la creacion del usuario. Arriba

Guardar usuario

Seleccione este botón si desea guardar los datos del usuario. Arriba
